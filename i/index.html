# 보유기술

Javascript ⭐️ ⭐️ ⭐️ 🌟

프론트엔드 개발자를 지향하고 있습니다
vue.js를 기반으로 코딩 스타일을 읽히고 있습니다
제가 배우는 javascript는 vue.js 공식 홈페이지의 스타일 가이드를 따르고 있습니다
예를들어 세미콜론(;)을 사용하지 않는다거나, 2회 이상의 공백 불허, camel, pascal case등
명명규칙 등이 있습니다
javascript는 개발에 기준이 되는 가장 중요하게 생각하는 언어 입니다

CSS ⭐️ ⭐️

디자인을 잘 만드는 것. 역시 프론트엔드 개발자의 중요한 요소라 생각하고 있습니다
scss, bootstrap 공식 홈페이지를 교과서 처럼 참조하고 있습니다
보기 좋게 만드는 것, 사용지로 하여금 익숙한 UX를 제공해 주는 것은
코드를 잘 짜는것 만큼 중요한 요소라는 생각을 갖고 있습니다

Java ⭐️ ⭐️

자바라는 언어는 제가 처음으로 배우게 된 프로그래밍 언어 입니다
객체지향의 개념을 잘 이해하도록 해준 언어이며, 현재까지 백엔드를 구현에 사용하는
가장 오랜기간 사용하고 있는 언어 입니다.
spring boot를 활용하는 백엔드 구축을 주로 진행하고 있습니다
개인적으로 프론트엔드 개발을 지향 하기에 자바에 대한 깊은 지식은 부족하다고 생각 합니다

DataBase ⭐️ ⭐️ ⭐️

개발 경력이 시작됨과 동시에 DB는 절대 떼어 놓을 수 없는 존재 같습니다
sql-server, oracle, mysql, postgresql, ...등 다양한 DB를 경험 했습니다
기초적인 PL/SQL 작성법과 trigger, scheduler, db-link를 활용 할 수 있습니다

Server ⭐️

서버환경을 구축하는 것은 정말 중요 하지만..
개인적으로 가장 자신이 없는 분야 입니다
Docker나 jenkins, Apache와 같은 서버환경 구축은 경험이 많이 부족한 편 입니다
주로 tmax-jeus, weblogic was 환경에서 개발을 진행 했습니다
